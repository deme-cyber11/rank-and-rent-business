### Research Phase

**Keyword Research**
```
Find high-value keywords for [NICHE] in [CITY], [STATE].
Include search volume, competition, and commercial intent analysis.
```

**Competitor Analysis**
```
Analyze top 10 competitors ranking for "[KEYWORD]".
Evaluate SEO strategies, content, backlinks, and weaknesses.
```

**SERP Analysis**
```
Analyze search results for "[KEYWORD]" and identify
what's ranking and why. Find opportunities to outrank.
```

**Content Gap Analysis**
```
Find content gaps and opportunities for [NICHE] in [CITY]
that competitors are missing.
```

### Build Phase

**Build Complete Site**
```
Build a rank and rent website for [NICHE] in [CITY] using the
templates. Target keywords: [LIST]. Include all pages, schema
markup, and on-page SEO.
```

**Write Homepage**
```
Write an SEO-optimized homepage for [BUSINESS NAME] targeting
"[PRIMARY KEYWORD]". Use the homepage template with local [CITY]
references.
```

**Write Services Page**
```
Create a comprehensive services page for [BUSINESS NAME].
Target keywords: [LIST]. Use the services page template.
```

**Generate Meta Tags**
```
Create optimized title tags and meta descriptions for all
pages on my [NICHE] site in [CITY].
```

**Generate Schema Markup**
```
Generate schema markup for [BUSINESS NAME], a [NICHE] business
in [CITY]. Include LocalBusiness, Service, and GeoCircle schemas.
```

**Optimize for AI (GEO)**
```
Optimize the content on [PAGE] for AI citation by ChatGPT,
Claude, and other AI systems. Make it quotable and fact-rich.
```

---

### Optimize Phase

**On-Page SEO Audit**
```
Audit the on-page SEO of [URL or site description].
Check all elements and provide optimization recommendations.
```

**Technical SEO Check**
```
Check the technical SEO of my [NICHE] site. Analyze page speed,
mobile responsiveness, schema, sitemap, and technical factors.
```

**Internal Linking**
```
Analyze and optimize the internal linking structure for
[SITE]. Recommend link opportunities and anchor text.
```

**Content Refresh**
```
Analyze [SITE or URL] and recommend content that should be
refreshed or updated. Focus on declining rankings.
```

---

### Monitor Phase

**Rank Tracking**
```
Set up rank tracking for these keywords on [SITE]: [LIST].
Track weekly and alert for changes > 5 positions.
```

**Backlink Analysis**
```
Analyze the backlink profile for [SITE or DOMAIN].
Find opportunities and toxic links.
```

**Performance Report**
```
Generate a monthly performance report for [SITE] including
rankings, traffic, leads, and recommendations.
```

**Set Up Alerts**
```
Set up ranking alerts for [SITE] on these keywords: [LIST].
Notify me of drops > 5 positions or page 1 losses.
```

**Service Page**

## üí° Quick Tips

### Best Niches
1. Emergency Plumbing ($1,500-$3,000/mo)
2. HVAC Repair ($1,500-$3,000/mo)
3. Roofing ($2,000-$4,000/mo)
4. Locksmith ($800-$2,000/mo)
5. Garage Doors ($1,000-$2,000/mo)

### Ideal Market Size
- Population: 50,000 - 500,000
- Not major metros (easier to rank)
- Growing economy
- Limited strong competition

### Target Keywords
- Search volume: 100-1,000/month
- Keyword difficulty: < 40
- Local intent (city name or "near me")
- Commercial/transactional intent

### Pricing Strategy
- **Fixed Rent**: $500-$3,000/month
- **Per Lead**: $25-$150/lead
- **Revenue Share**: 10-20% of closed deals
- **Hybrid**: Base rent + per-lead bonus

### Timeline Expectations
- Build: 1-2 weeks
- First rankings: 2-3 months
- Top 5 rankings: 3-6 months
- Ready to rent: 4-6 months
- Break even: 1-3 months after renting

## üîë SEO Best Practices

### Content
- Homepage: 1,000-1,500 words
- Services: 2,000-3,000 words
- Blog posts: 1,500-2,000 words
- Use primary keyword in H1, first paragraph, 2-3x naturally

### Technical
- Page speed > 85 (mobile & desktop)
- Mobile responsive
- HTTPS (SSL)
- Clean URL structure
- XML sitemap
- Schema markup

### Local SEO
- Google Business Profile
- Citations (20-30 directories)
- NAP consistency (Name, Address, Phone)
- Local content and references
- Reviews (when have client)

### Link Building
- Start with citations
- Local business directories
- Guest posts on local blogs
- Chamber of Commerce
- Industry associations
- 5-10 links per month minimum
