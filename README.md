# Rank & Rent Business - Claude Code Project

A comprehensive rank and rent business operation system powered by Claude Code and specialized SEO/GEO skills.

## Project Overview

This project provides a complete system for building, optimizing, and managing rank and rent websites using AI-powered automation through Claude Code.

### What is Rank & Rent?

Rank & Rent is a digital real estate business model where you:
1. Build and rank local service websites for specific geo-targeted keywords
2. Rent the leads or website to local businesses who need customers
3. Generate recurring monthly income from each ranked property

## Methodology: Research → Build → Optimize → Monitor

This project follows a proven four-phase approach:

### 1. Research Phase
- **Keyword Research**: Find high-value, low-competition local service keywords
- **Competitor Analysis**: Analyze what's ranking in your target markets
- **SERP Analysis**: Understand search intent and ranking factors
- **Content Gap Analysis**: Identify opportunities competitors miss

### 2. Build Phase
- **SEO Content Writing**: Create ranking-focused content
- **GEO Content Optimization**: Optimize for AI citation (ChatGPT, Claude, etc.)
- **Meta Tags Optimization**: Compelling titles and descriptions
- **Schema Markup**: Rich snippets for better visibility

### 3. Optimize Phase
- **On-Page SEO Audits**: Fix on-page issues
- **Technical SEO Checks**: Solve technical problems
- **Internal Linking**: Optimize site architecture
- **Content Refresh**: Keep content updated and relevant

### 4. Monitor Phase
- **Rank Tracking**: Monitor keyword positions
- **Backlink Analysis**: Build authority over time
- **Performance Reports**: Track ROI and metrics
- **Alert Management**: Catch ranking drops early

## Best Practices

### For Local SEO
- Focus on "city + service" keywords (e.g., "plumber miami")
- Build local citations (Google Business Profile, Yelp, etc.)
- Get local backlinks from chambers of commerce, local blogs
- Use location-specific schema markup
- Create location pages for multiple cities

### For Rank & Rent Success
- Choose services with high customer lifetime value (plumbing, roofing, HVAC)
- Target cities with population 50K-500K (less competition than major metros)
- Build multiple properties in the same niche for economies of scale
- Track metrics: rankings, traffic, leads, conversion rates
- Document everything for easier sales

### For GEO (AI Engine Optimization)
- Include clear definitions and FAQs
- Use specific data and statistics
- Structure content with Q&A format
- Keep information factually accurate
- Update content regularly
